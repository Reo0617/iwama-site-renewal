# Design System Skill

このskillはデザイン統一と再利用性向上を目的とする。

## 主な観点

- ボタン統一
- 色統一
- spacing統一
- typography統一
- component化
- section再利用
- UI一貫性

## 再利用対象

- CTAボタン
- GoogleMap導線
- Instagram導線
- LINE導線
- フッター
- Heroセクション
- Menuセクション

## 重視

- 保守しやすさ
- 再利用しやすさ
- 飲食店案件へ流用しやすい構成

## NG

- ページごとに別デザイン
- 重複CSS
- 無秩序なclass命名
