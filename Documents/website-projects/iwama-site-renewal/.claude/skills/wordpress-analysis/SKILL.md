# WordPress Analysis Skill

このskillは既存WordPressサイトの解析を目的とする。

## 主な観点

- 使用テーマ特定
- template構成解析
- CSS構造解析
- JS構造解析
- plugin依存確認
- page template確認
- functions.php確認
- トップページ構成解析

## 作業ルール

- いきなり大量編集しない
- まず構造理解を優先
- 編集対象ファイルを明示
- 既存機能を壊さない
- 保守性を重視

## NG

- functions.phpへの大量追記
- 不要なplugin追加
- WordPressコア編集
- 本番環境直接編集
