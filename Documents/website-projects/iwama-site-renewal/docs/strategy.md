# リニューアル戦略

## 戦術の核

PHPを触らず **CSSオーバーレイで「清潔感・高級感」を作る**。
最小リスク・最高保守性で15万円予算に収める。

## ビジュアル

- 黒地 + 白文字、赤/金アクセント
- 写真をヒーローで大きく
- 余白を倍増させて既存の「雑さ」を解消

## 導線改善

- スマホ下部に固定CTA（地図 / 電話 / LINE）
- SNSはQR依存をやめ、タップ可能なボタンに
- 導線優先度: GoogleMap > 電話 > LINE > Instagram

## 段階

1. 現状監査 & デザイン方向決定（アクセス不要・先行可能）
2. CSSで全体トーン刷新
3. トップページ集中改善
4. スマホQA
5. 納品

## 最大の関門

WP環境アクセス（管理画面 / FTP / 親テーマソース）が未確認。
これが無いと段階2以降が進まない。

- 着手前にクライアントへアクセス確認を依頼するのが最初のアクション
- 未確定の間は段階1（監査・参考分析・方向決定）まで先行可能
- 代替: 管理画面のみ取得できれば「追加CSS（カスタマイザ）」での適用も可能
